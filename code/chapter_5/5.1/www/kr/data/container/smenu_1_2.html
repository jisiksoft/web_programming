소프트웨어 개발


OS와 프로그래밍 언어에 독립적
- 다양한 소프트웨어를 개발하며 OS에 상관없이 모든 환경에서 개발이 가능
- 특정 Platform에 종속되지 않고 목적에 적합한 소프트웨어를 개발
- 현존하는 모든 프로그래밍 언어를 사용하여 개발이 가능

윈도우 기반
- Microsoft Windows API/MFC 응용프로그램을 개발
- Microsoft .Net Framework 프로그램을 개발

유닉스/리눅스 기반
- 서버 시스템 프로그램 개발 및 다양한 알고리즘을 구현

모바일 기반
- iPhone/Android 어플리케이션을 개발
- Native-App과 Hybrid-App 등 모든 환경에서 개발이 가능

Web 프로그램
- 최신 기술을 사용하여 인터넷에서의 홈페이지를 개발
- HTML5를 이용한 Web Application 개발이 가능
- 브라우저에서 화려한 UI의 구현이 가능하고, 세련된 Web 프로그램을 개발

디자인 전문가와 협업
- 전문 디자이너와 협업을 통해 세련된 디자인의 소프트웨어를 개발한다.
- 지식소프트는 소프트웨어 기술 개발에 역량을 집중한다.


* 현재 지식소프트 홈페이지는 디자이너가 아닌 지식소프트 김세훈 부장이 혼자서 개발 했기 때문에 화려하지는 않습니다. 김세훈 부장은 프로그래머가 디자인까지 잘 할 수 없기에 세상은 공정하다고 생각하며 열심히 일하고 있습니다.^^
